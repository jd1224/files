script takes an input file in csv format from the same folder as the program
which is named input.csv. file must be comma seperated with no spaces to 
work correctly with the script.  script writes data pulled from the 
greynoise api's ip search to a file in the same folder named output.csv.
user must have read and write permission for the folder in which the script
is run. script requires python and requests module installed. which can be 
done with pip install requests. script must be run from the directory
where requests module is found.
***************designed and run on ubuntu linux server core*****************
Resident folder is /usr/local/lib/python3.5/dist-packages
